A React application that generates random movie recommendations by genre. It demonstrates asynchronous API orchestration by chaining data from TMDB and OMDb to sync movie details with real-time IMDb ratings.

Technical Features

API Chaining: Integrates TMDB for discovery and OMDb for metadata.

State Management: Uses React Hooks for loading states and data fetching.

Security: Implements environment variables for API key protection.

Setup:

Clone the repository.

Add your REACT_APP_TMDB_KEY and REACT_APP_OMDB_KEY to a .env file.

Run npm install and npm start.
